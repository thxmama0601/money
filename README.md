# 好運搖獎機 🎰

威力彩・大樂透選號小工具，純靜態網頁，可直接部署到 GitHub Pages。

## 功能

- 威力彩（38 選 6 + 第二區 1~8）與大樂透（49 選 6 + 特別號）雙模式
- 搖獎機動畫電腦選號（使用 `crypto.getRandomValues` 產生均勻隨機數）
- 1 對 1 手動選號盤
- 威力彩近 10 期冷熱號統計（2026/05/28–06/29，純娛樂參考）

## 誠實聲明

每期開獎皆為獨立隨機事件，本工具**無法**也**不可能**提高中獎機率。
冷熱號統計沒有預測能力，任何組合的頭獎機率皆相同。
未滿 18 歲不得購買或兌領彩券，投注請適度。

## 部署到 GitHub Pages

1. 把 `index.html` 和本檔案上傳到儲存庫根目錄
2. 進入 Settings → Pages
3. Source 選 `Deploy from a branch`，Branch 選 `main`、資料夾選 `/ (root)`，按 Save
4. 等一兩分鐘，網站就會出現在 `https://<你的帳號>.github.io/money/`
